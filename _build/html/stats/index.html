Stats¶

Stats - Graphite Interface¶

class stats.stats.Stats(app, name=None)¶

Stats objects send messages to a Graphite server and retrieve graphs.

message(name, value, datetime=None)¶

Contruct a Graphite message from name, value and time If time is omitted, the current time will be used.

now None¶

Return the current time (in Unix time).

publish(iterator)¶

Publish all stats in iterator that returns (value, datetime) tuples.

unix_time(datetime)¶

Convert a datetime to Unix time.
