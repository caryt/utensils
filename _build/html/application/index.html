Configuration Options¶

class application.config.Config¶

Config objects store configuration options. Default values for each option are loaded from defaults.py, and can be overridden by setting a corresponding environment variable.

defaults()¶

Read the default configuration options from defaults.py.

html()¶

Return an HTML representation of this configuration.

json()¶

Return a list representation of the conf, suitable for JSON encoding.

load_script(script)¶

Load environment variables from a bash shell script.

overrides()¶

Override each config from a matching environment variable, if it exists.
